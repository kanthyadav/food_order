# 🍔 Food Order Platform

A full-stack MERN Food Ordering Platform with AI-powered food recommendations using Google Gemini.

## 🚀 Live Demo

* Frontend: https://food-order-8qe3.vercel.app
* Backend: https://food-order-eyxp.onrender.com

## ✨ Features

### Customer

* User Registration & Login
* Browse Restaurants
* Search Food Items
* Add to Cart
* Place Orders
* Track Order Status

### Restaurant Owner

* Apply for Restaurant Ownership
* Admin Approval Required
* Add Food Items
* Manage Restaurant Menu
* Receive Orders
* Update Order Status

### Admin

* Dashboard Statistics
* Approve Restaurant Owners
* Reject Restaurant Requests
* Generate Restaurants using AI
* Monitor Platform Activity

### AI Assistant

* Food Recommendations
* Budget-based Suggestions
* Restaurant Suggestions
* Gemini AI Integration

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router
* Axios
* CSS

### Backend

* Node.js
* Express.js
* JWT Authentication

### Database

* MongoDB Atlas
* Mongoose

### AI

* Google Gemini API

### Deployment

* Vercel
* Render
